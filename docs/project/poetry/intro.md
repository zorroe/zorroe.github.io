# 一个古诗网站

> 一个使用Nuxt3和springboot开发的前后端分离古诗网站。
>
> 项目地址：[https://github.com/zorroe/ancient-poetry](https://github.com/zorroe/ancient-poetry)

## 技术栈

- 前端：Nuxt3
- 后端：Spring Boot
- 数据库：MySQL
- 搜索引擎：Elasticsearch

## 功能

- 搜索古诗
- 根据朝代浏览古诗
