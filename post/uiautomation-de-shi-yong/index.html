UIAutomation的使用

环境准备

python版本：3.7.3

uiautomation版本：2.0.17

获取win元素工具：UI Spy

如果是Windows XP系统，请确保系统目录有这个文件：UIAutomationCore.dll。如果没有，需要安装补丁 KB971513 才能支持UIAutomtion.

代码示例

import subprocess
import time
import uiautomation as auto
auto.uiautomation.SetGlobalSearchTimeout(15)  # 设置全局搜索超时 15


def main():

    def GetFirstChild(control):
        return control.GetFirstChildControl()

    def GetNextSibling(control):
        return control.GetNextSiblingControl()

    subprocess.Popen("D:\\Tencent\\QQ\\Bin\\QQ.exe")
    desktop = auto.GetRootControl()  # 查找桌面控件
    toolBar = desktop.ToolBarControl(ClassName="ToolbarWindow32",Name="用户提示通知区域")  # 工具栏控件
    while(True):
        flag = False
        # 遍历工具栏控件，找到登录成功的QQ控件
        for control, depth in auto.WalkTree(toolBar, getFirstChild=GetFirstChild, getNextSibling=GetNextSibling, includeTop=False, maxDepth=1):
            if(str(control.Name).startswith("QQ") and str(control.Name) is not "QQ"):
                control.Click()
                flag = True
        if(flag):
            break
        time.sleep(0.5)

    qqWindow = auto.WindowControl(searchDepth=1,ClassName="TXGuiFoundation",Name="QQ")  # 查找QQ界面
    qqWindow.ListItemControl(searchDepth=18,Name="我的Android手机").DoubleClick()  # 点击聊天
    time.sleep(1)

    chatWindow = auto.WindowControl(searchDepth=1,ClassName="TXGuiFoundation",Name="我的Android手机")  # 查找聊天界面
    chatWindow.EditControl(searchDepth=11,foundIndex=1).SendKeys("hello android手机")  # 聊天框输入文字
    time.sleep(1)
    chatWindow.ButtonControl(searchDepth=12,Name="发送(&S)").Click()  # 发送信息

if __name__ == '__main__':
    main()

查找控件的可选参数

  • searchFromControl = None：从哪个控件开始查找，如果为None，从根节点Desktop开始查找
  • searchDepth = 0xFFFFFFFF：搜索深度
  • searchInterval = SEARCH_INTERVAL：搜索间隔
  • foundIndex = 1 ：搜索到的满足搜索条件的控件索引，索引从1开始
  • Name ：控件名字
  • SubName ：控件部分名字
  • RegexName ：使用re.match匹配符合正则表达式的名字，Name,SubName,RegexName只能使用一个，不能同时使用
  • ClassName：类名字
  • AutomationId：控件AutomationId
  • ControlType：控件类型
  • Depth：控件相对于searchFromControl的精确深度
  • Compare：自定义比较函数function(control: Control, depth: int)->bool

searchDepth和Depth的区别：

  • searchDepth在指定的深度范围内（包括1-searchDepth层中的所有子孙控件）搜索第一个满足搜索条件的控件
  • Depth只在Depth所在的深度（如果Depth>1，排除1~searchDepth-1层中的所有子孙控件）搜索第一个满足搜索条件的控件
