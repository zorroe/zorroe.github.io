Mock.js

前言

在前端开发的时候，往往都需要都需要后端提供接口数据来进行相应页面的开发。但这个时候就需要后端先行开发，之后前端才进行开发，从而可能会影响开发效率。

那前端能不能独立进行开发呢？答案是可以的。接下来就有请我们今天的主角，Mock.js。

Mock.js

  • 是一款模拟数据生成器
  • 生成随机数据，拦截Ajax请求
  • Mock.js这个工具完全可以通过我们设置的规则，来提供数据，从而解决前端不能独立开发的问题
  • 对接后端接口的时候，也不需要修改既有代码

基本使用

  1. 生成随机字符串

    var Mock = require('mockjs')

const data = Mock.mock({
    'title': '@ctitle(1,10)', // 生成中文标题，长度为1-10个字符
    'words|1-10': '@cword()', // 生成中文字符串，长度为1-10个字符
})

console.log(data)

// { title: '据再达场价是回', words: '利才质件' }

  2. 生成图片

    const img = Mock.mock({
    img_url: "@image('100x100', '#fff', '#000', 'png','坤坤')",
})

console.log(img)

// { img_url: 'http://dummyimage.com/100x100/fff/000.png&text=坤坤' }

  3. 生成日期

    const date = Mock.mock({
    date: '@date("yyyy-MM-dd HH:mm:ss")',
})

console.log(date)

// { date: '1995-01-18 14:11:26' }

  4. 生成json

    const list = Mock.mock({
    list: [
        {
            id: '@increment(1)',
            name: '@cname(2,4)',
            age: '@integer(18,30)',
            'gender|1': ['男', '女'],
        },
    ],
})

console.log(list.list)

// [ { id: 1, name: '卢丽', age: 28, gender: '男' } ]

  5. 生成json列表

    const list = Mock.mock({
    'list|5': [
        {
            id: '@increment(1)',     // 自增类型
            name: '@cname(2,4)',     // 中文姓名
            age: '@integer(18,30)',  // 整数
            'gender|1': ['男', '女'],// 枚举
        },
    ],
})

console.log(list.list)

// mock/index.ts
const list = Mock.mock({
    'list|5': [
        {
            id: '@increment(1)',
            title: '@title(2,4)',
            img_url: `@image('100x100', '#fff', '#000', 'png','坤坤')`,
            add_time: '@date("yyyy-MM-dd HH:mm:ss")',
        },
    ],
})

// mock/index.ts
Mock.mock(RegExp('/api/delete/news'), 'delete', (option: any) => {
    // 获取url的参数
    const id = option.url.split('?')[1].split('=')[1]
    // 删除对应id的数据
    list.list = list.list.filter((item: any) => item.id != id)
    console.log(list.list)
    return {
        code: 200,
        msg: 'success',
    }
})

// App.vue

// mock/index.ts
Mock.mock('/api/put/news', 'put', (option: any) => {
    const news = JSON.parse(option.body)
    // 根据id修改
    list.list = list.list.map((item: any) => {
        if (item.id == news.id) {
            item.title = news.title
            item.img_url = news.img_url
            item.add_time = news.add_time
        }
        return item
    })
})

// App.vue

    getTableData()
  })
